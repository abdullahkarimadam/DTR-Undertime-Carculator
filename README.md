# DTR Undertime Calculator (Philippines)

A simple, modern, and mobile-friendly Daily Time Record (DTR) undertime calculator designed for Filipino employees. Automatically computes undertime based on your scheduled time (AM/PM) and flags Philippine regular/special holidays.

### Features:
- Customizable work schedule (AM in/out, PM in/out)
- Multi-user support
- Automatic Philippine holidays (regular & movable dates)
- Add custom company holidays
- Visual monthly calendar with color-coded undertime & holidays
- Fully responsive – works perfectly on mobile (iPhone & Android)
- No database required (uses JSON file)
- Clean, minimalist, and modern UI (Tailwind CSS)

Perfect for personal use or small teams who want to track undertime easily and accurately.

Made with PHP • Responsive • Pinoy-made
